# Posts service
* No db is used. data would be temporarly stored in a variable.
* It will have two requests

## post request
* path '/posts'
* method POST
* body { title: string }
* goal create a new post

## get request
* path '/posts'
* method GET
* body -
* goal retrive all posts

## To run a service
* npm start
* Posts service will run on 4000 port.
